# Atividade Prática - Frameworks para Web Front-End e Back-End

Este repositório contém o código-fonte desenvolvido para a atividade prática da disciplina de **Frameworks para Web Front-End**, integrando tecnologias de Front-End e Back-End.

O projeto consiste na instalação, configuração básica e execução de uma aplicação utilizando **Next.js** e **Django**.

---

## 🚀 Tecnologias Utilizadas

### Front-End
* **Next.js** (Framework React)
* HTML5 (Tag `<h2>` para exibição de identificação)
* Node.js & npm

### Back-End
* **Django** (Framework Python)
* HTML5 (Tag `<footer>` para direitos autorais)
* SQLite (Banco de dados padrão)
